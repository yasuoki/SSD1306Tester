# SSD1306Tester
SSD1306 OLED Display test program for Aergia Devices 

## 使用方法

1. RP2040 Zeroにプログラムをインストールします。

2. RP2040 Zeroのピンを以下のようにSSD1306ディスプレイモジュールに接続します。
```
   RP2040           SSD1306  
     0 <-------------> SDA  
     1 <-------------> SCL  
  3.3V <-------------> VCC  
   GND <-------------> GND  
```

3. RP2040 ZeroをUSB電源に接続します
4. LEDが赤青色に点滅し、OLDディスプレイにテスト文字が表示されます。
